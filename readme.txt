== Description ==

This plugin generates news letter for using short code and saves subscriptions to backend database.  Just use short code [aparnf-reg-form] on post or pages.




== Installation ==

1= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'apa-register-newsletter-form.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard

= Using FTP =

1. Extract 'apa-register-newsletter-form.zip' to your computer
1. Upload the 'apa-register-newsletter-form' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Display the 'Reg List ' into a menu in WordPress.
1. you can see alld data, registration summery and download csv
